NZGrapher Locally Hosted
You can download the files for NZGrapher from http://www.jake4maths.com/grapher.zip
You will need to extract them into a folder on a web server running PHP (with the GD library enabled).

Full details are inside the README file.

Custom Folders
Teachers can upload files into the foldertemplate folder (which you can rename to whatever you would like by just changing the name of the folder directly on the server... or copy it as many times as you would like) and then teachers can access it and add files by going to http://url-to-access-nzgrapher/foldertemplate/
The password for this folder is inside a file in the folder.
Students can then access the data in that folder by going to http://url-to-access-nzgrapher/?folder=foldertemplate
Many schools have multiple folders going (level1, level2, level3 etc)
If you have any questions don't hesitate to email me.
